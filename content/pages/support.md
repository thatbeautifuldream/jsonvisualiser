# Support

If you need help with JSON Visualiser or the browser extension, email:

[support@jsonvisualiser.com](mailto:support@jsonvisualiser.com)

Please include:

- A short description of the issue
- The page or feature you were using
- Steps to reproduce the problem
- Screenshots or sample JSON if relevant and safe to share

We will review your message and get back to you as soon as possible.
