# Privacy Policy for JSON Visualiser

Last updated: March 13, 2026

This Privacy Policy explains how JSON Visualiser, including the website at `jsonvisualiser.com` and related browser extension experiences, handles information.

JSON Visualiser is a tool for viewing, validating, formatting, and exploring JSON data. We aim to keep the product simple and privacy-conscious.

## Scope

This policy applies to:

- The JSON Visualiser website
- The `/extension` page used with the browser extension workflow
- Browser storage used to support the service

## Information We Process

### JSON Content You Choose to Load

When you paste, edit, or load JSON into JSON Visualiser, that content is processed in your browser so the app can render the editor, tree view, graph view, validation results, and formatting tools.

If you use the browser extension workflow, the extension may pass JSON content and the source page URL into the JSON Visualiser extension page so the content can be displayed and explored.

### Page or Source URL

When JSON is loaded through the extension workflow, JSON Visualiser may store the source URL as metadata associated with the current document in the browser. This is used to identify where the loaded JSON came from inside the current workspace.

### Browser Storage Data

JSON Visualiser uses browser storage to persist workspace state and improve continuity between refreshes and tabs. Depending on the feature being used, this may include:

- JSON content saved in IndexedDB
- A generated tab identifier in session storage
- Local storage markers used for tab lifecycle cleanup

This storage is used to restore your workspace and manage tab-specific state. It is stored in your browser environment.

## How We Use Information

We use information to:

- Render and operate the JSON editor and visualisation tools
- Validate, format, and display JSON content
- Persist workspace state in your browser
- Support the browser extension workflow
- Diagnose bugs, abuse, and operational issues

## How Information Is Shared

We do not sell personal data.

We may also disclose information if required by law, legal process, or to protect rights, safety, and the integrity of the service.

## Remote Processing and Extension Use

If you use the browser extension workflow, JSON content is displayed inside a hosted JSON Visualiser page. That means the website code loaded in your browser handles the content needed to render the interface.

If you want to avoid sending data into a hosted page experience, do not use the extension workflow for sensitive content.

## Data Retention

Browser-stored workspace data remains in your browser until it is cleared, replaced, or removed by cleanup logic. Some tab state may be automatically cleaned up after inactivity or when tabs close.

## Your Choices

You can:

- Avoid entering sensitive information into JSON Visualiser
- Clear browser storage for the site from your browser settings
- Stop using the extension workflow for content you do not want handled in the hosted page

## Children

JSON Visualiser is not directed to children under 13, and we do not knowingly collect personal information from children.

## Changes to This Policy

We may update this Privacy Policy from time to time. When we do, we will update the "Last updated" date on this page.
